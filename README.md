# Shell-Scripting-Project
Debit &amp; Credit Transaction Process on ATM Managment System
🏧 ATM Management System – Shell Script Project
📌 Project Description

This project is a simple ATM Management System developed using Bash Shell Scripting.
It simulates basic banking operations like:

Account Creation

ATM Card Processing

PIN Verification

Cash Withdrawal

Cash Deposit

This project was created as part of my Shell Scripting practice and academic learning.

🎯 Features

Create new customer account

Validate ID proof (Aadhar / PAN / License)

Select account type (Savings / Current)

Initial deposit during account creation

Apply for ATM card

PIN verification system

Cash Withdrawal (multiple of 100 rule)

Cash Deposit (multiple of 500 rule)

Balance checking after every transaction

Option to perform multiple transactions

🛠 Technologies Used

Bash Shell Scripting

Linux Operating System

Basic Linux Commands

Conditional Statements (if, case)

Loops (while)

Functions in Bash

Arithmetic Operations in Shell

Input Validation using Regular Expressions
